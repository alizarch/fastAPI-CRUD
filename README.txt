Steps to configure and test using uvicore
*****************************************************************************
Step 1 : Make Envinonment  ($ python -m venv env)
Step 2 : Install requinments.txt ($ pip install -r requirements.txt)
step 3 : Run server on cmd (uvicorn main:app --reload)
Step 4 : Go to fastAPI - Swagger UI (http://127.0.0.1:8000/docs).
Step 5 : Please read documentation to test APIs on Swagger UI.
